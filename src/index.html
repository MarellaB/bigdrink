What Do I Do?

Install: npm install -g tbd-cli

Use: drink @username-<package-name>

And you're done! All dependencies are installed and you are ready to begin.

How do I submit my gulpfile?

Create a repository on Github labeled
tbg-<package-name>

Add your gulpfile.js

Then add a .runtime file

And you're done!

What is TBD?

TBD (The Big Drink) is npm for Gulp files.

To keep the namespace clear for Gulp I have created TBD to allow people to join and share common Gulp files.

I felt this was easier and more convenient, allowing people who live and breath Gulp to create the optimized convenient task runners we all love, and easily share them with the world.

With the ability to set npm install commands and have all projects ready to go with a single drink TBD will help remove the the time wasted spent writing a gulp file for every project.
